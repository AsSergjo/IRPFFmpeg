# Changelog

Все заметные изменения проекта можно вести в этом файле.

Формат основан на идее Keep a Changelog, версии можно отмечать тегами GitHub Releases.

## Unreleased

- Добавлена система локализации интерфейса с выбором русского или английского языка.
- Добавлен `language_manager.cpp/.h` для загрузки `.lng` файлов и сохранения выбранного языка.
- Добавлены файлы локализации `Language/english.lng` и `Language/russian.lng` в репозиторий и релизный архив.
- Добавлено всплывающее окно с обложкой и названием трека при смене композиции в режиме трея.
- Добавлена настройка включения/отключения всплывающей обложки.
- Добавлено сохранение позиции всплывающего окна между запусками.
- Обновлен внешний вид слайдеров.
- Обновлен скриншот главного окна.
- Уточнен список обязательных runtime DLL в README и указано, где их получить.
- Уточнена сборочная настройка DLL: зависимости должны попадать в `heap_dll`, а релизный комплект должен использовать проверенный набор x64 DLL.
- Подготовлена GitHub-документация проекта.
- Добавлены инструкции по сборке, релизу, архитектуре и использованию.
- Добавлены `.gitignore`, `.gitattributes`, issue templates и security policy.
- Добавлены MIT License для кода проекта и notices для сторонних компонентов.
- Уточнено авторство AsSergjo и добавлена просьба об указании автора при использовании кода.
