# Руководство пользователя

## Запуск

Запускайте приложение через:

```text
Start_IRPFFmpeg.exe
```

Загрузчик проверит наличие `IRPFFmpeg.exe`, создаст папку `heap_dll` при необходимости, перенесет DLL из корня релизной папки в `heap_dll`, добавит эту папку во временный путь поиска библиотек и запустит основное приложение.

## Плейлист

Основной список станций хранится в `playlist.m3u`. Его можно редактировать вручную или менять из интерфейса приложения.

В контекстном меню списка станций можно добавить станцию, изменить ее название, сохранить текущее отображаемое название в `playlist.m3u` или удалить станцию. Если поток передает ICY-название станции, IRPFFmpeg может временно показать его в списке; ручное сохранение фиксирует название и отключает дальнейшую автоматическую замену для этой станции.

Поддерживаемые URL обычно начинаются с:

```text
http://
https://
icy://
mms://
rtsp://
```

Пример M3U:

```text
#EXTM3U
#EXTINF:-1,Example Radio
https://example.com/stream
```

## Воспроизведение

Приложение умеет:

- запускать и останавливать поток;
- переключаться между станциями;
- возвращаться к предыдущей станции;
- показывать текущий трек, если станция передает метаданные;
- показывать название станции из ICY metadata, если оно доступно;
- показывать технический статус потока;
- сохранять состояние между запусками.

Если станция не отвечает или поток временно недоступен, приложение пытается переподключиться.

Если сервер присылает битые аудиоданные, приложение может временно показать статус `Пропускаем битые данные от сервера...`. В этом состоянии оно пропускает некорректные фреймы и ждет восстановления стабильного потока.

## Обложки

IRPFFmpeg пытается найти обложку текущего трека, загрузить ее и сохранить в `cover_cache`. Кэш нужен, чтобы не скачивать одно и то же изображение повторно.

Если обложка недоступна, воспроизведение продолжится без нее.

## Запись эфира

Записи сохраняются в папку `Rec`.

Доступны форматы:

- MP3 320 kbit/sec;
- FLAC 16-bit.

Имя файла формируется из текущего трека, если поток передает корректные метаданные. Недопустимые для Windows символы удаляются или заменяются.

Если обложка и метаданные доступны, приложение пытается добавить их в записанный файл.

## Звук

Доступные настройки:

- язык интерфейса;
- громкость;
- бас;
- верхние частоты;
- Stereo Width;
- Exciter;
- DeepBass;
- Dynamic Auto Volume;
- LUFS-нормализация станций;
- GainRider;
- финальный лимитер.

Dynamic Auto Volume и GainRider полезны на станциях с неровной громкостью. LUFS-нормализация станций медленно приводит уровень разных потоков к единому ориентиру `-8.10 LUFS`, чтобы переключение между станциями было комфортнее. Stereo Width, Exciter и DeepBass лучше использовать умеренно: слишком большие значения могут сделать звук менее естественным.

Язык интерфейса можно выбрать в настройках. Файлы локализации лежат в папке `Language` рядом с exe, а выбранный язык сохраняется между запусками в `app.dat`.

## Системный трей

Приложение может сворачиваться в системный трей. Это удобно, если радио должно продолжать играть в фоне.

Если включена настройка показа обложки при смене трека, во время работы из трея приложение показывает небольшое всплывающее окно с текущей обложкой и названием композиции. Окно можно перетащить в удобное место; позиция сохраняется между запусками.

## Если приложение не запускается

Проверьте:

1. Запускается именно `Start_IRPFFmpeg.exe`.
2. Рядом есть `IRPFFmpeg.exe`.
3. Рядом есть папка `heap_dll`.
4. В `heap_dll` лежат FFmpeg, SDL2 и сопутствующие DLL.
5. Папка приложения доступна на запись, если нужны настройки, записи и кэш обложек.
