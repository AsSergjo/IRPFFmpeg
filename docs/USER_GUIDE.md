# Руководство пользователя

## Запуск

Запускайте приложение через:

```text
Start_IRPFFmpeg.exe
```

Загрузчик проверит наличие `IRPFFmpeg.exe`, создаст папку `heap_dll` при необходимости, перенесет DLL из корня релизной папки в `heap_dll`, добавит эту папку во временный путь поиска библиотек и запустит основное приложение.

## Плейлист

Основной список станций хранится в `playlist.m3u`. Его можно редактировать вручную или менять из интерфейса приложения.

Поддерживаемые URL обычно начинаются с:

```text
http://
https://
icy://
mms://
rtsp://
```

Пример M3U:

```text
#EXTM3U
#EXTINF:-1,Example Radio
https://example.com/stream
```

## Воспроизведение

Приложение умеет:

- запускать и останавливать поток;
- переключаться между станциями;
- возвращаться к предыдущей станции;
- показывать текущий трек, если станция передает метаданные;
- показывать технический статус потока;
- сохранять состояние между запусками.

Если станция не отвечает или поток временно недоступен, приложение пытается переподключиться.

## Обложки

IRPFFmpeg пытается найти обложку текущего трека, загрузить ее и сохранить в `cover_cache`. Кэш нужен, чтобы не скачивать одно и то же изображение повторно.

Если обложка недоступна, воспроизведение продолжится без нее.

## Запись эфира

Записи сохраняются в папку `Rec`.

Доступны форматы:

- MP3 320 kbit/sec;
- FLAC 16-bit.

Имя файла формируется из текущего трека, если поток передает корректные метаданные. Недопустимые для Windows символы удаляются или заменяются.

Если обложка и метаданные доступны, приложение пытается добавить их в записанный файл.

## Звук

Доступные настройки:

- громкость;
- бас;
- верхние частоты;
- Stereo Width;
- Exciter;
- DeepBass;
- Dynamic Auto Volume;
- GainRider;
- финальный лимитер.

Dynamic Auto Volume и GainRider полезны на станциях с неровной громкостью. Stereo Width, Exciter и DeepBass лучше использовать умеренно: слишком большие значения могут сделать звук менее естественным.

## Системный трей

Приложение может сворачиваться в системный трей. Это удобно, если радио должно продолжать играть в фоне.

## Если приложение не запускается

Проверьте:

1. Запускается именно `Start_IRPFFmpeg.exe`.
2. Рядом есть `IRPFFmpeg.exe`.
3. Рядом есть папка `heap_dll`.
4. В `heap_dll` лежат FFmpeg, SDL2 и сопутствующие DLL.
5. Папка приложения доступна на запись, если нужны настройки, записи и кэш обложек.
