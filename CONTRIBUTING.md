# Участие в разработке

Спасибо за интерес к IRPFFmpeg.

## Как подготовить окружение

1. Установите Visual Studio 2022 с MSVC v143 и Windows SDK 10.
2. Подготовьте FFmpeg development package.
3. Подготовьте SDL2 и SDL2_image.
4. Обновите пути к зависимостям, если они отличаются от локальных путей автора.
5. Соберите `Release|x64` или `Debug|x64`.

Подробности находятся в [docs/BUILD.md](docs/BUILD.md).

## Перед отправкой изменений

- Не добавляйте в коммит `x64/`, `.vs/`, `.pdb`, `.obj`, `app.dat`, `Rec/`, `cover_cache/` и DLL.
- Проверяйте запуск через `Start_IRPFFmpeg.exe`.
- Для изменений в воспроизведении проверяйте несколько разных потоков.
- Для изменений в записи проверяйте MP3 и FLAC.
- Для изменений в UI проверяйте, что окно корректно открывается и закрывается, а трей не ломает повторный запуск.

## Стиль

Проект использует C++17 и Win32 API. Новые изменения лучше делать без крупных сторонних фреймворков, если задача решается существующей архитектурой.

Комментарии полезны там, где логика неочевидна: аудиообработка, кодировки, потоки, FFmpeg lifecycle.
